# 多商户SaaS版扫码点餐系统（移动H5端）

### 介绍
（H5端）多商户SaaS版扫码点餐系统，支持后台点餐、多人同时在线点餐、购物车共享、餐桌状态实时监控，菜品管理、餐桌管理等众多功能。
